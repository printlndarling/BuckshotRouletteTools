# 恶魔轮盘助手 🎯

一个帮助《恶魔轮盘》(Buckshot Roulette) 玩家追踪子弹状态和分析概率的实用工具。

## 功能特性 ✨

### 🎮 游戏设置
- **灵活配置**：支持自定义总子弹数量和真弹/假弹分配
- **智能验证**：自动确保输入数据的有效性
- **快速开始**：一键开始新的游戏追踪

### 📊 实时追踪
- **子弹状态可视化**：直观显示弹舱中每发子弹的状态
  - 🔴 红色：真弹
  - 🔵 蓝色：假弹
  - ⚫ 灰色：未知
- **实时统计**：显示剩余真弹、假弹和总子弹数量
- **消耗记录**：快速记录射出的子弹类型

### 📈 概率分析
- **总体概率**：显示剩余子弹中真弹/假弹的比例
- **下一发预测**：计算下一发子弹的概率分布
- **各位置分析**：分析每个位置子弹的概率情况
- **动态更新**：根据已知信息实时调整概率

### 🔧 道具集成
- **放大镜**：记录已知的下一发子弹类型
- **手机**：记录特定位置的已知子弹类型
- **智能概率重算**：根据道具信息自动更新概率分布

## 使用方法 📖

### 1. 游戏设置
1. 打开工具，在设置面板输入：
   - 总子弹数量
   - 真弹数量
   - 假弹数量
2. 点击"开始游戏"

### 2. 游戏进行中
- **射出子弹**：点击相应按钮记录射出的子弹类型
- **使用道具**：点击道具按钮记录已知信息
- **查看概率**：实时查看概率分析和弹舱状态

### 3. 重新开始
点击"重新开始"按钮可以重置工具，开始新的游戏追踪。
